# Use GitHub

## <u>Branch setup:</u>

### Generate branch:

git branch #name#

### Delete branch:

git branch -D #name#

### Move through branches:

- git checkout #name#
- git switch #name#

### Get main changes to branch

(be in the branch you need)

git pull origin main

### Push branch

git add .

git commit -m "..."

git push origin #name#

### Merge branch

(be in main branch)

git merge #name# main
